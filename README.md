# Pleer - Медиаплеер для Android TV

**Мощный и гибкий медиаплеер для Android TV приставок с поддержкой Torrserv, Lampa и всех современных кодеков.**

## 🎬 Основные возможности

- ✅ Поддержка всех популярных видеокодеков (H.264, H.265/HEVC, VP8, VP9, AV1)
- ✅ Поддержка аудиокодеков (AAC, MP3, FLAC, Opus, Vorbis, AC3, DTS)
- ✅ Интеграция с Torrserv для прямого доступа к торрент-потокам
- ✅ Интеграция с Lampa для синхронизации
- ✅ Красивый интерфейс, оптимизированный для TV
- ✅ Управление с пульта ДУ Android TV
- ✅ Настройка скорости воспроизведения (0.5x - 2.0x)
- ✅ Отображение буфера и скорости загрузки
- ✅ Функции перемотки (±10 сек)
- ✅ Сохранение настроек и истории
- ✅ Поддержка HLS и DASH потоков
- ✅ Кастомизируемый интерфейс

## 📋 Системные требования

- Android 5.0 (API 21) и выше
- Android TV рекомендуется
- 2 GB RAM минимум
- 100 MB свободного места

## 🚀 Установка на Android TV через iPhone

### Способ 1: Использование ADB (рекомендуется)

#### Шаг 1: Подготовка
1. На iPhone скачайте приложение **"ADB Manager"** из App Store
2. На Android TV приставке перейдите в **Настройки > О телевизоре** и кликните 7 раз на **"Build Number"**, чтобы включить режим разработчика
3. Перейдите в **Настройки > Параметры разработчика** и включите **ADB**
4. Запишите IP-адрес приставки (обычно видно в сетевых настройках)

#### Шаг 2: Подключение через iPhone
1. Откройте ADB Manager на iPhone
2. Введите IP-адрес Android TV: `192.168.X.X:5555` (замените на свой)
3. Нажмите "Connect"
4. Подтвердите подключение на Android TV

#### Шаг 3: Установка APK
1. Скачайте APK файл приложения (через браузер на iPhone)
2. В ADB Manager выберите **"Install APK"**
3. Выберите скачанный APK файл
4. Дождитесь завершения установки

### Способ 2: Через файловый менеджер

1. На iPhone в iCloud Drive загрузите APK файл
2. На Android TV откройте файловый менеджер
3. Перейдите в папку с загруженным файлом
4. Нажмите на APK файл и выберите "Установить"
5. Подтвердите установку

### Способ 3: Через Google Play (если приложение опубликовано)

1. Откройте Play Market на Android TV
2. Найдите "Pleer"
3. Нажмите "Установить"
4. Дождитесь завершения

## ⚙️ Первоначальная настройка

### Конфигурирование Torrserv

1. Откройте приложение Pleer
2. Нажмите **Меню > Настройки**
3. Введите адрес вашего Torrserv сервера:
   - По умолчанию: `http://localhost:8090`
   - Если на другой машине: `http://192.168.X.X:8090`
4. Сохраните настройки

### Конфигурирование Lampa

1. В **Настройках** приложения найдите "Lampa URL"
2. Введите адрес Lampa сервера:
   - По умолчанию: `http://localhost:8085`
3. Сохраните

## 🎮 Управление плеером

### С пульта ДУ

| Кнопка | Действие |
|--------|----------|
| ◄ (влево) | Перемотка назад на 10 сек |
| ► (вправо) | Перемотка вперед на 10 сек |
| ▲ (вверх) | Увеличить громкость |
| ▼ (вниз) | Уменьшить громкость |
| OK / Enter | Play/Pause |
| ⏯ (Play) | Начать воспроизведение |
| ⏸ (Pause) | Пауза |
| Назад | Вернуться в меню |

### На экране

- **Browse** - открыть браузер Torrserv
- **Play/Pause** - начать/остановить воспроизведение
- **-10s / +10s** - быстрая перемотка
- **Скорость (0.5x - 2.0x)** - изменить скорость воспроизведения
- **Settings** - открыть настройки

## 🔍 Использование браузера Torrserv

1. Нажмите кнопку **"Browse"** на главном экране
2. Введите название фильма или сериала в поисковую строку
3. Нажмите **"Search"**
4. Выберите нужный результат из списка
5. Нажмите на карточку потока
6. Плеер автоматически начнет воспроизведение

## 🎨 Настройки качества

### Буфер видео

- **Низкий** (50 MB) - для медленного интернета
- **Средний** (100 MB) - рекомендуется
- **Высокий** (200 MB) - для стабильного соединения

### Параметры воспроизведения

- **Скорость** - 0.5x до 2.0x
- **Автоматическое продолжение** - продолжить с последней позиции
- **Скрытие элементов управления** - автоматически через 5 сек

## 🔧 Продвинутые настройки

### Кастомная IP адресация

Если ваши сервисы находятся на разных машинах:

1. Откройте **Настройки > Сервисы**
2. Измените адреса:
   - Torrserv: `http://YOUR_IP:8090`
   - Lampa: `http://YOUR_IP:8085`

### Поддерживаемые форматы контейнеров

- **Видео**: MP4, MKV, WebM, AVI, MOV, FLV, TS, M3U8 (HLS), MPD (DASH)
- **Аудио**: MP3, AAC, FLAC, Opus, OGG

## 🐛 Решение проблем

### Плеер не находит Torrserv

1. Проверьте, что Torrserv запущен на вашей машине
2. Убедитесь, что Android TV и компьютер в одной сети WiFi
3. Проверьте правильность IP адреса:
   ```bash
   # На компьютере в терминале
   ipconfig (Windows) или ifconfig (Mac/Linux)
   ```
4. Проверьте, что порт 8090 открыт

### Видео тормозит или буферизируется

1. Проверьте скорость интернета
2. Уменьшите качество видео
3. Закройте другие приложения
4. Выберите поток с большим количеством seeds
5. Увеличьте размер буфера в настройках

### Звук не работает

1. Проверьте громкость Android TV
2. Убедитесь, что аудиокодек поддерживается
3. Попробуйте другой источник видео

### Приложение падает при запуске

1. Очистите кэш приложения: **Настройки > Приложения > Pleer > Хранилище > Очистить кэш**
2. Переустановите приложение
3. Проверьте, что у вас достаточно свободной памяти

## 📦 Сборка из исходных кодов

### Требования

- Android Studio 2022.1+
- JDK 11+
- Android SDK 34
- Gradle 8.0+

### Процесс сборки

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/AvedEd/pleer.git
   cd pleer
   ```

2. Откройте проект в Android Studio

3. Дождитесь синхронизации Gradle

4. Выполните сборку:
   ```bash
   ./gradlew assembleRelease
   ```

5. APK будет расположен в: `app/build/outputs/apk/release/app-release.apk`

## 🤝 Интеграция с вашими сервисами

### API для Torrserv

```
GET http://localhost:8090/api/search?query=фильм
GET http://localhost:8090/api/top?category=movies
```

### API для Lampa

```
POST http://localhost:8085/api/player/play
Body: {"url": "...", "title": "...", "player": "pleer"}

GET http://localhost:8085/api/player/status
```
