# 📱 ПОЛНАЯ ИНСТРУКЦИЯ УСТАНОВКИ НА ANDROID TV ЧЕРЕЗ iPhone

## 🎯 Быстрый старт (5 минут)

Если у вас нет времени - используйте этот способ:

### Шаг 1: Подготовка
1. **На Android TV приставке**:
   - Перейдите в **Параметры > Сведения об устройстве**
   - Кликните 7 раз на **"Build Number"**
   - Перейдите в **Параметры > Опции разработчика** (появятся новые опции)
   - Включите **"ADB по сети"** (или "ADB over Network")
   - Запишите IP-адрес (будет показан на экране)

2. **На iPhone**:
   - Откройте App Store
   - Найдите и установите **"ADB Manager"** или **"Android Control"**
   - Убедитесь, что iPhone подключен к той же WiFi, что и Android TV

### Шаг 2: Скачивание APK
1. На iPhone откройте Safari
2. Перейдите на GitHub: https://github.com/AvedEd/pleer/releases
3. Скачайте последний файл `pleer-release.apk`
4. Файл будет в папке **"Загрузки"** или **iCloud Drive**

### Шаг 3: Установка через ADB Manager
1. Откройте **ADB Manager** на iPhone
2. Нажмите **"+"** или **"Add Device"**
3. Введите IP-адрес Android TV: `192.168.X.X:5555` (замените X на ваши цифры)
4. Нажмите **"Connect"**
5. На Android TV появится запрос - нажмите **"Разрешить"**
6. В ADB Manager нажмите **"Install APK"**
7. Выберите скачанный файл `pleer-release.apk`
8. Дождитесь завершения установки

**Готово! Приложение установлено.** 🎉

## 🔧 ПОДРОБНАЯ ИНСТРУКЦИЯ (ВСЕ СПОСОБЫ)

### СПОСОБ 1: ADB Manager (РЕКОМЕНДУЕТСЯ) ⭐

#### Преимущества:
- ✓ Самый надежный способ
- ✓ Работает через WiFi
- ✓ Не требует кабелей
- ✓ Можно устанавливать обновления

#### Инструкция:

**Часть A: Подготовка Android TV**

1. На пульте нажмите **Home** (главная)
2. Перейдите в **Параметры** (обычно в левом углу)
3. Выберите **Сведения об устройстве** или **О телевизоре**
4. Найдите поле **"Build Number"** или **"Номер сборки"**
5. **Нажимайте на это поле 7 раз подряд** (пока не появится сообщение "Вы разработчик")
6. Вернитесь назад в Параметры
7. Должна появиться новая опция **"Опции разработчика"** или **"Developer Options"**
8. Откройте ее
9. Найдите и включите:
   - **"Отладка по USB"** (USB Debugging) - если есть
   - **"ADB по сети"** или **"ADB over Network"** - ВАМ НУЖНА ЭТА
10. Если нужна сетевая отладка - нажмите на нее, появится IP-адрес
11. **Запишите или запомните этот IP-адрес** (выглядит как `192.168.1.100` или похоже)

**Часть B: Подготовка iPhone**

1. Откройте **App Store** на iPhone
2. В поиске найдите **"ADB Manager"** (есть несколько, берите с наибольшим количеством загрузок)
3. Нажмите **"Получить"** > **"Установить"** > введите Face ID / Touch ID
4. Дождитесь установки
5. Откройте приложение ADB Manager
6. **Важно**: Убедитесь, что iPhone подключен к **ТОЙ ЖЕ WiFi СЕТИ**, что и Android TV!

**Часть C: Подключение**

1. В ADB Manager нажмите кнопку **"+"** или **"Add New Device"**
2. В появившемся поле введите:
   ```
   192.168.1.100:5555
   ```
   (замените цифры на ваш IP-адрес из шага 11 части A)
3. Нажмите **"Connect"** или **"Подключиться"**
4. **На экране Android TV появится запрос разрешения!**
5. На пульте перейдите на **"Разрешить"** и нажмите **OK**
6. В ADB Manager должно появиться сообщение "Connected" (зелёная галочка)

**Часть D: Скачивание и установка APK**

1. На iPhone откройте **Safari**
2. Перейдите на сайт: `https://github.com/AvedEd/pleer/releases`
3. Найдите последний релиз (самый верхний)
4. Нажмите на файл **`pleer-release.apk`** или **`app-release.apk`**
5. Нажмите кнопку скачивания ⬇️
6. Выберите **"Копировать ссылку"** или **"Download"**
7. Дождитесь загрузки (файл обычно 50-80 MB)
8. Вернитесь в **ADB Manager**
9. Нажмите кнопку **"Install APK"** или **"📁 Установить приложение"**
10. Выберите скачанный файл из **"Загрузок"** или **iCloud Drive**
11. Нажмите **"Open"** или **"Открыть"**
12. **Подождите 1-2 минуты** - приложение будет устанавливаться
13. Когда появится сообщение **"Install Complete"** - приложение установлено! ✓

**Часть E: Запуск приложения**

1. На пульте Android TV нажмите **Home**
2. Найдите приложение **"Pleer"** (обычно оно автоматически добавляется в меню)
3. Нажмите на него - приложение запустится 🚀

## ⚙️ ПЕРВОНАЧАЛЬНАЯ НАСТРОЙКА

### После первого запуска:

1. **Приложение откроется с главным экраном**
2. Нажмите на кнопку **"Settings"** или **"⚙ Настройки"** (если видите управление)
3. **Настройте Torrserv:**
   - Поле: `Torrserv Server URL`
   - Введите: `http://192.168.X.X:8090` (замените X на IP вашего компьютера, где запущен Torrserv)
   - Если Torrserv на **локальной машине**: `http://localhost:8090`

4. **Настройте Lampa (если используется):**
   - Поле: `Lampa Server URL`
   - Введите: `http://192.168.X.X:8085` (аналогично выше)

5. **Настройте буфер:**
   - Поле: `Buffer Size (MB)`
   - Для медленного интернета: `50-100`
   - Для нормального: `100-150`
   - Для быстрого: `150-200`

6. Нажмите **"Save"** (Сохранить)

## 🔍 ПОИСК IP-АДРЕСА КОМПЬЮТЕРА

**Если вы не знаете IP-адрес своего компьютера, где запущен Torrserv:**

### Windows:
1. Откройте **Command Prompt** (Командную строку)
2. Введите: `ipconfig`
3. Найдите строку **"IPv4 Address"** - это ваш IP
4. Обычно это выглядит как: `192.168.X.X`

### Mac:
1. Откройте **Terminal**
2. Введите: `ifconfig`
3. Найдите строку **"inet"** (под вашей сетью WiFi)
4. Это ваш IP-адрес

### Linux:
1. Откройте терминал
2. Введите: `hostname -I`
3. Увидите ваш IP

## 🐛 РЕШЕНИЕ ЧАСТЫХ ПРОБЛЕМ

### "Не могу подключиться к Torrserv"

✓ Проверьте:
1. Запущен ли Torrserv на компьютере?
2. Находятся ли оба устройства (TV и компьютер) в одной WiFi?
3. Правильный ли IP-адрес? (проверьте по инструкции выше)
4. Открыт ли порт 8090 в брандмауэре?

✓ Решение:
```bash
# На компьютере Windows - проверить в командной строке:
netstat -an | findstr :8090

# На Mac/Linux:
lsof -i :8090
```

Если ничего не выводится - Torrserv не запущен. Запустите его.

### "Видео воспроизводится с задержками / буферизируется"

✓ Решения:
1. Вы��ерите поток с большим количеством **seeds** (больше людей = быстрее)
2. Увеличьте размер буфера в настройках до 200 MB
3. Проверьте скорость интернета (https://speedtest.net)
4. Закройте другие приложения на TV
5. Попробуйте использовать Ethernet кабель вместо WiFi

### "Приложение не установилось через ADB Manager"

✓ Попробуйте:
1. Перезагрузите оба устройства (TV и iPhone)
2. Отключитесь и подключитесь заново в ADB Manager
3. Используйте Способ 2 (через файловый менеджер)
4. Проверьте, что свободно минимум 200 MB на TV

### "На экране ничего не видно / черный экран"

✓ Решения:
1. Перезагрузите приложение
2. Проверьте, что подключение к Torrserv установлено
3. Попробуйте поиск н��вого видео
4. Очистите кэш приложения: **Параметры > Приложения > Pleer > Хранилище > Очистить кэш**

### "Звук не работает"

✓ Проверьте:
1. Не отключена ли громкость на TV?
2. Не отключена ли громкость в самом приложении?
3. Поддерживает ли TV аудиокодек видео?
4. Попробуйте другой источник видео

## 🚀 РАСШИРЕННЫЕ СОВЕТЫ

### Для лучшего опыта:

1. **Используйте Ethernet**: если возможно, подключите TV кабелем вместо WiFi - это дает стабильность

2. **Оптимальный размер буфера**:
   - Медленный интернет: 50 MB
   - Средний: 100 MB (рекомендуется)
   - Быстрый: 150-200 MB

3. **Выбирайте потоки с рейтингом**: ищите потоки с максимальным количеством seeds и минимальным peers

4. **Закрывайте фоновые приложения**: особенно приложения, которые используют интернет

5. **Используйте горячие клавиши пульта** для быстрого управления:
   - Долгое нажатие Play = список качества
   - Долгое нажатие Left/Right = перемотка на 30 сек
